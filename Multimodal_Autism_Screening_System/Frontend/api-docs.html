📖 API Documentation

Complete REST API reference for Multimodal ASD Screening System

Base URL

http://localhost:5000

Production: Replace with your deployed URL

Authentication

Currently, the API does not require authentication. Future versions will include JWT token-based authentication.

📡 Endpoints

GET /

Description: Health check endpoint to verify API is running

Request

curl http://localhost:5000/

Response (200 OK)

{ "status": "online", "message": "Multimodal ASD Screening API", "version": "1.0.0", "endpoints": [ "/", "/api/models/status", "/api/predict" ] }

GET /api/models/status

Description: Check which AI models are loaded and ready

Request

curl http://localhost:5000/api/models/status

Response (200 OK)

{ "behavioral_model": true, "voice_model": true, "facial_model": true, "fusion_model": true, "all_models_loaded": true, "timestamp": "2025-11-17T00:49:00Z" }

Error Response (500)

{ "error": "Models not loaded", "details": "Behavioral model missing" }

POST /api/predict

Description: Make ASD prediction from multimodal data

Request Parameters (multipart/form-data)

Parameter Type Required Description
behavioral_data File (CSV) REQUIRED Behavioral questionnaire responses (20 features)
voice_data File (WAV/MP3) REQUIRED Voice recording (3-10 seconds recommended)
image_data File (JPG/PNG) REQUIRED Frontal facial image (224×224 or higher)

Example Request (cURL)

curl -X POST http://localhost:5000/api/predict \ -F "behavioral_data=@behavioral.csv" \ -F "voice_data=@voice_sample.wav" \ -F "image_data=@face_image.jpg"

Example Request (Python)

import requests url = "http://localhost:5000/api/predict" files = { 'behavioral_data': open('behavioral.csv', 'rb'), 'voice_data': open('voice_sample.wav', 'rb'), 'image_data': open('face_image.jpg', 'rb') } response = requests.post(url, files=files) print(response.json())

Example Request (JavaScript/Fetch)

const formData = new FormData(); formData.append('behavioral_data', behavioralFile); formData.append('voice_data', voiceFile); formData.append('image_data', imageFile); fetch('http://localhost:5000/api/predict', { method: 'POST', body: formData }) .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error));

Success Response (200 OK)

{ "prediction": "ASD", "confidence": 0.93, "risk_level": "High", "individual_scores": { "behavioral": 0.87, "voice": 0.84, "facial": 0.90, "fusion": 0.93 }, "interpretation": { "result": "ASD Detected", "recommendation": "Professional clinical evaluation recommended", "accuracy": "93%" }, "timestamp": "2025-11-17T00:49:00Z", "processing_time_ms": 2340 }

Error Responses

400 Bad Request - Missing Files { "error": "Missing required files", "missing": ["behavioral_data", "voice_data"] }
415 Unsupported Media Type { "error": "Invalid file format", "details": "Behavioral data must be CSV" }
500 Internal Server Error { "error": "Prediction failed", "details": "Model inference error" }

📊 Response Codes

Code Status Description
200 OK Request successful
400 Bad Request Missing or invalid parameters
415 Unsupported Media Type Invalid file format
500 Internal Server Error Server-side processing error

📄 CSV Format Example

Behavioral data CSV should have 20 feature columns:

feature_0,feature_1,feature_2,...,feature_19 0.5,0.8,0.3,...,0.7
